Suggested Naming Convention
````````````````````````````

    Components:
        Case   : PascalCase
        Format : [FileName].js 
        Eg: Splash.js

    Services, Providers, and other:
        Case   : kebab-case
        Format : [name]-[context].[extension]
        Eg: login-service.js
    
    It is suggested to use these file naming conventions to quickly identify the file type in context,
        which will be helpful for everyone
